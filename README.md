Project Description

The Voice-Driven Code Generation System is a software solution designed to simplify the programming process by allowing users to generate code using natural language commands. Instead of manually writing code with strict syntax, users can provide instructions in simple language, making programming more accessible and user-friendly.

The system works by processing user input through multiple stages inspired by compiler design. Initially, the input is normalized to remove unnecessary words and standardize the format. It is then analyzed to extract the user’s intent using pattern matching techniques. Based on the identified intent, the system performs semantic validation using a symbol table to ensure that variables are correctly declared and used.

After successful validation, the system generates structured and syntactically correct code in the C programming language. The generated code includes proper formatting, indentation, and basic constructs such as variable declaration, assignment, loops, and print statements. The final output is stored in a source file, which can be compiled and executed.

The project combines concepts from natural language processing and compiler design to create a reliable and efficient code generation system. It focuses on reducing syntax errors, improving productivity, and making coding easier for beginners.
